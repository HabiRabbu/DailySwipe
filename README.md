# DailySwipe

A SocialMedia style News Reading App!

- Saves your name and preferred newspaper location data to SharedPreferences
- Horizontal "Swipe" style interface to browse through articles
- Swipe up to view the article
- Menu button at the top right to change your newspaper location (China unfortunately doesn't have any articles available.)

Uses MediaStack API on a free trial - This will run out and cause the app to stop displaying articles at some point.
